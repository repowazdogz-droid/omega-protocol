# TEMPLATE EVALUATION CHECKLIST (MUST PASS ALL)

Use this checklist before declaring any template "done".

## A) TRUST & SAFETY
- [ ] On-screen text clearly states what this IS
- [ ] On-screen text clearly states what this IS NOT
- [ ] No operational control or instructions
- [ ] No recommendations, "best", or optimization language
- [ ] No autonomy, no auto-advance, no simulation

## B) INTERACTION DISCIPLINE
- [ ] Camera only (orbit/pan/zoom)
- [ ] Layer toggles only (no manipulation)
- [ ] Inspector shows neutral metadata
- [ ] Reset View resets camera only
- [ ] No hidden interactions

## C) LAYER GRAMMAR
- [ ] STRUCTURE present
- [ ] CONSTRAINTS present
- [ ] UNCERTAINTY present
- [ ] ENVELOPE present (if applicable)
- [ ] Layers can be toggled independently

## D) INSPECTOR QUALITY
- [ ] Every selectable object is tagged
- [ ] Inspector shows:
      - name
      - layer
      - status (Known / Assumed / Unknown / Conservative)
      - "what this is NOT"
- [ ] No advice or interpretation in Inspector copy

## E) CALM TEST
- [ ] No animation except camera
- [ ] No urgency or gamification
- [ ] Feels legible and non-impressive
- [ ] Does not feel like a simulator

## F) EXPLAINABILITY
- [ ] Can explain purpose in <60 seconds
- [ ] Can explain limits without defensiveness
- [ ] Can explain uncertainty without apology
